# Installation

## Compiling from C++

On Mac:

~~~~
brew install gsl
brew install boost
brew install htslib
brew install pkgconfig
make
~~~~

### Testing

~~~~
npm install
make test
~~~~

## Installing via npm

The software is compiled to WebAssembly as [machineboss](https://www.npmjs.com/package/machineboss) on npm

~~~~
npm install machineboss
~~~~
